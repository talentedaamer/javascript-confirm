Welcome to javascript-confirm

JavaScript Confirm is a lightweight, customizable library that replaces the default browser confirmation dialog with a sleek, modern UI. It’s easy to integrate into any project and doesn’t require jQuery, making it ideal for minimalistic setups. Compatible with AMD, CommonJS, and browser globals.

Getting Started

To get started with JavaScript Confirm, follow these steps:

  1. Include the Library: Add the javascript-confirm.js file to your project.
  2. Include Styles: Make sure to include javascript-confirm.css in your project for the default styles.

Installation

You can install JavaScript Confirm via npm:

npm install javascript-confirm

Or, if you prefer, you can include it directly in your project using a script tag:

<script src="path/to/javascript-confirm.js"></script>
<link rel="stylesheet" href="path/to/javascript-confirm.css">

Usage

To use JavaScript Confirm, call the javascriptConfirm method on any DOM element. For example:

<button id="myButton">Click me</button>
          <script>
          document.getElementById('myButton').javascriptConfirm({
              title: 'Confirm Action',
              message: 'Are you sure you want to proceed?',
              onConfirm: function() {
                  alert('Confirmed!');
              },
              onCancel: function() {
                  alert('Cancelled!');
              }
          });
          </script>

Options

You can customize the confirmation dialog using the following options:

  • title: The title of the confirmation dialog (default: 'Dialog').
  • message: The message displayed in the dialog (default: 'Are you sure?').
  • confirmText: The text for the confirm button (default: 'Yes').
  • cancelText: The text for the cancel button (default: 'No').
  • onConfirm: Callback function when the user confirms (default: function() {}).
  • onCancel: Callback function when the user cancels (default: function() {}).

Customization

You can customize the appearance of the confirmation dialog by overriding the default styles. The default styles are included in javascript-confirm.css. You can create your own CSS rules to match your design preferences.

/* Example custom styles */
          .javascript-confirm-dialog {
              background-color: #f9f9f9;
              border: 2px solid #ccc;
          }
          .javascript-button.button-primary {
              background-color: #007bff;
              color: #fff;
          }

Themes

To apply different themes, you can create CSS classes that override the default theme styles. Attach these classes to the dialog elements or apply them globally to style the confirmation dialogs according to your design system.

/* Example theme */
          .theme-dark .javascript-confirm-dialog {
              background-color: #333;
              color: #fff;
          }
          .theme-dark .javascript-button {
              background-color: #555;
              color: #fff;
          }

API

Element.prototype.javascriptConfirm(options)

Extends the Element prototype to add the javascriptConfirm method.

Parameters

  • options: An object with the following properties:
    • title: (string) Title of the dialog.
    • message: (string) Message to display in the dialog.
    • confirmText: (string) Text for the confirm button.
    • cancelText: (string) Text for the cancel button.
    • onConfirm: (function) Callback function when the user confirms.
    • onCancel: (function) Callback function when the user cancels.

Example

document.getElementById('myButton').javascriptConfirm({
                  title: 'Confirm Action',
                  message: 'Are you sure you want to proceed?',
                  onConfirm: function() {
                      console.log('Confirmed!');
                  },
                  onCancel: function() {
                      console.log('Cancelled!');
                  }
              });
